🧠 Description

This project explores multimodal machine learning across three interconnected tasks: image-based multi-label genre classification, natural language processing for text-based genre prediction, and graph neural networks for link prediction. The goal is to investigate how visual, textual, and relational data can be jointly leveraged to improve learning performance in complex recommendation and classification scenarios.

⚙️ Project Overview

The work is divided into three main components:

Multi-Label Image Classification: Genre prediction from movie posters using visual features
Natural Language Processing (NLP): Genre classification based on textual information such as descriptions and metadata
Graph Neural Networks (GNNs): Link prediction to model relationships between movies, users, or entities
